# Água Mineral Campo Grande

Site estático para distribuição de água mineral em Campo Grande/MS.

## Domínio
- **Produção:** https://aguamineralcampogrande.com.br

## Stack
- HTML puro (sem frameworks)
- CSS inline com variáveis
- Hospedagem: Cloudflare Pages
- Deploy automático via push
